# Housing Price Prediction

This project uses machine learning to predict housing prices based on various features like area, number of bedrooms, bathrooms, etc.


## Data

The dataset contains various features that may influence house prices. Key features include:
- **area**: Square footage of the property
- **bedrooms**: Number of bedrooms
- **bathrooms**: Number of bathrooms
- **stories**: Number of floors in the house
- **mainroad**: Accessibility to a main road (yes or no)
- **guestroom**: Presence of a guest room (yes or no)
- **basement**: Presence of a basement (yes or no)
- **hotwaterheating**: Availability of hot water heating (yes or no)
- **airconditioning**: Presence of air conditioning (yes or no)
- **parking**: Number of parking spaces
- **prefarea**: Preferred area (yes or no)
- **furnishingstatus**: Furnishing status of the house (furnished, semi-furnished, or unfurnished)
